# Hostel Management System

## Description
This project is developed using PHP and MySQL to manage hostel students, complaints, and administration.

## Features
- Admin Panel
- Complaint Management
- Student Records

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## How to Run
1. Copy project to htdocs folder
2. Start Apache & MySQL in XAMPP
3. Import database (.sql file)
4. Open browser: http://localhost/hostel
